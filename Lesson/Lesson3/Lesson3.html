Lesson 3 : Arrays




What is Array?

An array is a linear data structure which can store a fixed-size sequential collection of elements of the same type, which are stored together in a contiguous block of memory. Each element in the array can be accessed by its index, which is an integer value that specifies its position within the array.

All arrays consist of contiguous memory locations. The lowest address corresponds to the first element and the highest address to the last element.



Declaring Arrays

To declare array in C, define the data type (like int) and specify the name of the array followed by square brackets [ ].



Initializing Arrays

Initializing an array in computer programming means assigning initial values to the elements of the array. The number of values between braces {} shall not be greater than the number of elements in the array. In C, there are different ways to initialize arrays. Here are some examples.

1. Initializing an array with a specific values.
        int arr[] = {1, 2, 3, 4, 5};
2. Initializing an array with values at declaration:
        int arr[10] = {0};
3. Initializing an array with a specific size and values
       int balance[5] = {1000, 2, 3, 7, 50};



Accessing Array Elements


An element is accessed by indexing the array name. This is done by placing the index of the element within square brackets after the name of the array. To access an array element, it follows a method:

array_name[index];

  • Accessing the first element of an array: arr[0]
  • Accessing the second element of an array: arr[1]



    • Types of Array


    1. One – Dimensional Array

    One-dimensional array is a data structure that stores a collection of elements of the same data type in a sequential manner. The elements in a one-dimensional array are accessed using an index that starts from zero and goes up to one less than the size of the array.

    2. Multi-dimensional Array

    A multi-dimensional array is an array that contains other arrays as its elements. In other words, it is an array of arrays. The most common type of multi-dimensional array is a two-dimensional array, which is often used to represent matrices or tables of data.

    To declare 2D array, just write:

                  int myArray[m][n];



    3. Three – dimensional Array

    A three-dimensional array, also known as a 3D array, is a data structure that consists of a collection of elements organized in a three-dimensional grid or matrix. It is similar to a two-dimensional array, but with an additional dimension.

    To declare 3D array, just write;

                  int myArray[m][n][o];
