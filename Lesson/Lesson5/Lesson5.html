Lesson 5: Structures in C



In the world of C programming, structures come to the rescue when you want to bring different data pieces together in a neat package. It's like putting together puzzle pieces that don't have to be the same shape or size. In this lesson, we'll dive into structures and learn how to create them, access their parts, handle arrays of structures, and even nest them within each other. Let's embark on our journey to structure mastery!




Defining Structures: Creating Your Custom Data Types

Imagine you're a master architect, and structures are your blueprints. To define a structure, you'll use the struct keyword, followed by the structure's name and a list of its member variables enclosed in curly braces.

struct Employee {

char name[50];
int age;
float salary;

};

In our case, we've designed a structure named Employee, which consists of three members: name, age, and salary.



Initializing Structures: Setting Up the Perfect Data Foundation

Let's simplify the process of setting up structures by initializing them right from the start. It's like magically preparing everything before you even start building.

struct Employee emp = {"John", 25, 5000.0};

You can also initialize individual members using the member access operator.

struct Employee emp;
emp.name = "John";
emp.age = 25;
emp.salary = 5000.0;



Arrays of Structures: Grouping Data for Efficient Management

Imagine you're managing a group of employees, and you want to keep track of all their details. Arrays of structures come to the rescue!

struct Employee employees[10];

In this example, we create an array called employees that can store up to 10 Employee records.

To access the members of individual structures in the array, you can use the index notation.

employees[0].name = "John";
employees[0].age = 25;
employees[0].salary = 5000.0;



Nested Structures: Building Towers of Data

Sometimes, you need to go beyond simple structures and create more complex entities. Nesting structures allows you to build towers of data, where one structure holds another structure inside it.

struct Date {

int day;
int month;
int year;

};

struct Person {

char name[50];
struct Date birthdate;

};

In this example, we have a structure called Person that contains a nested structure called Date. It's like having a box within a box!

struct Person student;
student.birthdate.day = 15;
student.birthdate.month = 5;
student.birthdate.year = 2000;



Passing Structures to Functions: Sharing Data for Collaborative Work

Structures are meant to be shared! You can pass them to functions, either by copying their contents or by letting functions work directly with them using pointers.

Passing by value:

void display(struct Employee emp) {

printf("Name: %s\n", emp.name);
printf("Age: %d\n", emp.age);
printf("Salary: %.2f\n", emp.salary);

}


// Usage
display(emp);

Passing by value means that a copy of the structure is made and passed to the function. The display function takes an Employee structure as a parameter and prints its members.

Passing by reference:

void increaseSalary(struct Employee *emp) {

emp->salary *= 1.1; // Increase salary by 10%

}

// Usage
increaseSalary(&emp);

Passing by reference means that instead of copying the structure, we pass a pointer to the structure to the function. The increaseSalary function takes a pointer to an Employee structure and modifies the salary member using the -> operator.




Structures serve as building blocks for organizing and managing data in your programs. They allow you to create custom data types, access their members, work with arrays of structures, nest them within each other, and even share them with functions. With this newfound knowledge, you're now equipped to tackle more complex data organization challenges in your C programming journey. Happy coding!
